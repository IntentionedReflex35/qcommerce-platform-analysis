## Data Dictionary — `quick_commerce_data_raw.csv`

| Column                    | Type  | Description                        | Range / Values                                                                              |

| `Order_ID`                | int   | Unique order identifier            | 1000001 – 2000000                                                                           |

## Synthetic Data Note

This dataset was synthetically generated to reflect realistic Q-Commerce distributions. It was chosen to demonstrate the full analytical lifecycle
(auditing, cleaning, outlier treatment, EDA, statistical testing) rather than to draw real-world competitive conclusions. Correlation structure 
between variables is weaker than would be expected in production data.
