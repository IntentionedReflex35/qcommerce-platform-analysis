# Plots Directory

All figures are generated by running `notebooks/Analysis_Report.ipynb`.

| File                                | Description                                                                                            |
|-------------------------------------|--------------------------------------------------------------------------------------------------------|
| `fig_04_outlier_boxplots.png`       | IQR boxplots for the four continuous business metrics — shows outlier distribution before treatment    |

| `fig_05_2_age_distribution.png`     | Customer age range share overall and broken down by platform                                           |

| `fig_05_5_delivery_performance.png` | Average delivery time and delivery time consistency (std dev) ranked by platform                       |
| `fig_05_5b_rating_vs_time.png`      | Average delivery partner rating across delivery time bands — tests whether speed predicts satisfaction |
| `fig_05_6_discount_impact.png`      | Average order value and order volume for discounted vs non-discounted orders                           |
| `fig_06_efficiency_score.png`       | Composite operational efficiency ranking (order volume + delivery speed) across all platforms          |
| `fig_07_bulk_segment.png`           | Bulk buyer segment profile — platform share, discount usage, and order value vs delivery time          |
| `fig_08_correlation_matrix.png`     | Pearson correlation matrix across all continuous features in the clean dataset                         |
